Cómo añadir esta URL en tu programa de streaming

La URL es la misma para todos los programas — solo cambia la configuración de la fuente al pegarla.

TikTok LIVE Studio
  1. Abre TikTok LIVE Studio y pulsa Añadir Fuente → Enlace.
  2. Pega la URL copiada en el campo correspondiente.
  3. Activa el interruptor Resolución personalizada.
  4. Pon Ancho (W) = 780 y Alto (H) = 1920.
  5. Tras añadir la fuente, selecciónala en el lienzo y pulsa el icono Ajustar a la pantalla (segundo icono de la barra de herramientas de la fuente).
Por qué 780×1920 y no 1080×1920
Live Studio tiene un fallo conocido: si añades la fuente con Resolución personalizada 1080×1920 directamente, no se ajusta al lienzo y deja bandas. Con 780×1920 + Ajustar a la pantalla, Live Studio la escala correctamente.
OBS Studio / Streamlabs
  1. En tu escena, pulsa + → Navegador (Browser Source).
  2. Pega la URL en el campo URL.
  3. Pon Ancho = 1080 y Alto = 1920.
Otros programas (Meld Studio)

Cualquier programa con fuente tipo Navegador acepta esta URL. Configura el tamaño del Browser Source en 1080×1920.
