⛏️ Minecraft Command Builder Para configurar comandos complejos con todas las opciones (NBT, armadura, encantamientos, etc.), usa el constructor completo en:

TikControl → Acciones y Eventos → Acción Avanzada → Minecraft
Selecciona un comando previamente guardado en TikControl
Escribe comandos manualmente (uno por línea). Soporta {player}, {username}, {giftname}, etc.
Placeholders disponibles:
{player} - Nombre del usuario
{username} - Alias del usuario
{giftname} - Nombre del regalo
{amount} - Cantidad
{count} - Contador
